# Frontend Architecture

This section details the frontend architecture of the Poll Automation System.

## Core Structure

*   **Framework**: React with TypeScript
*   **State Management**: Redux Toolkit
*   **UI Components**: Shadcn UI (based on Radix UI)
*   **Styling**: Tailwind CSS


# Technical Stack

*   **Framework**: React
*   **Language**: TypeScript
*   **State Management**: Redux Toolkit
*   **UI Library**: Shadcn UI (Radix UI + Tailwind CSS)
*   **Data Visualization**: Chart.js (inferred from components)
*   **Form Handling**: React Hook Form (inferred from patterns)
*   **Build Tool**: Vite
*   **Linting**: ESLint
*   **Styling**: PostCSS
*   **Testing**: Jest (inferred from test files)
